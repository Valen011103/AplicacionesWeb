background Define el fondo de un elemento.

Sus variaciones y respectivas funciones:

background-color: Define el color de fondo de un elemento.

background-image: Permite establecer una imagen como fondo.

background-position: Define dónde se coloca la imagen dentro del contenedor.

background-size: Controla el tamaño de la imagen en relación con el elemento.

background-repeat: Especifica si la imagen debe repetirse o no.

background-attachment: Determina si el fondo se desplaza con el contenido o se mantiene fijo.

background-clip: Controla hasta dónde se extiende el fondo dentro del elemento.

background-origin: Define desde dónde se posiciona la imagen de fondo. 		background-color:
  • red (color por nombre)
  • #ff0000 (color en hexadecimal)
  • rgb(255, 0, 0) (color en RGB)
  • rrgba(255, 0, 0, 0.5) (color con transparencia)
background-image:
  • url('imagen.jpg') (Imagen de fondo)
  • linear-gradient(to right, red, blue)(Degradado)
background-position:
  • left top (Arriba a la izquierda)
  • center (Centro)
  • right bottom (Abajo a la derecha)
  • background-position: 50% 50%; (Coordenadas en porcentaje)
  • background-position: 100px 200px (Coordenadas en píxeles)
background-size:
  • auto (Tamaño original de la imagen )
  • cover (Cubre todo el elemento, sin distorsionar )
  • contain (Ajusta la imagen para que encaje completamente)
  • 100px 50px; (Ancho x Alto personalizado)
background-repeat:
  • repeat (Se repite en ambas direcciones)
  • no-repeat (No se repite)
  • repeat-x (Se repite solo en el eje X)
  • repeat-y (Se repite solo en el eje Y)
background-attachment:
  • scroll (Se mueve con el contenido)
  • fixed (Permanece fijo en la pantalla)
  • local (Se mueve dentro del contenedor)
background-clip:
  • border-box (Se extiende hasta el borde (por defecto)
  • padding-box () Se recorta en el área de padding)
  • content-box (Solo se muestra en el contenido)
background-origin:
  • border-box; (Desde el borde)
  • padding-box (Desde el padding)
  • content-box (Desde el contenido)

display Define el modelo de caja del elemento.

Controla si los elementos aparecen en línea o en bloque.

Permite estructuras avanzadas como Flexbox y Grid.

Puede ocultar elementos sin eliminarlos del código.
 none: Oculta el elemento.

block: El elemento ocupa todo el ancho disponible.

inline: El elemento se comporta en línea (no genera salto de línea).

inline-block: Se comporta como en línea, pero permite definir ancho y alto.

flex: Activa el modelo Flexbox.

grid: Activa el modelo Grid.

table: Se comporta como una tabla.

inherit: Hereda el valor del padre.
 .container { display: flex; }
justify-content

(Solo funciona en display: flex o display: grid)		 Controla la alineación horizontal de los elementos en un contenedor flex o grid. flex-start: Alinea los elementos al inicio.

flex-end: Alinea los elementos al final.

center: Centra los elementos.

space-between: Espacio uniforme entre elementos.

space-around: Espacio alrededor de cada elemento.

space-evenly: Espacio uniforme en todos lados.
 .container {

display: flex; justify-content: center;

}
padding Define el espacio interno entre el contenido y el borde del elemento. px: Unidad fija de píxeles en la pantalla.
Características:
  • Tiene un tamaño absoluto, no cambia con el contenedor
  • Se usa para tamaños de fuente, bordes, márgenes, etc
  • No es recomendable para diseños responsivos
%: Relativo al elemento padre.
Características:
  • Se adapta dinámicamente al tamaño del contenedor
  • Se usa para width, height, margin, padding, etc
em: tamaño de la fuente del elemento padre.
Características:
  • Útil para hacer que el texto escale con el diseño
  • Se usa en font-size, padding, margin, etc
  • Puede generar efectos de herencia acumulativa
rem: tamaño base del html (por defecto, 16px).
Características:
  • No hereda acumulativamente (más estable que em)
  • Se usa para font-size, padding, margin, etc.
  • No es recomendable para diseños responsivos
auto: El navegador decide el tamaño del elemento según el contenido o el espacio disponible.
Características:
  • Útil en width, height, margin
  • Se usa comúnmente en margin: auto; para centrar elementos

@import Permite cargar una hoja de estilo externa dentro de otra hoja CSS.

Se usa para organizar mejor los estilos y reutilizar código.

Se coloca al inicio del archivo CSS, antes de cualquier otra regla.
 importaciones de links, .css, librerías y más @import url('https://fonts.googleapis.com/css2?family=Poppins:wght@400;700&display=swap');
